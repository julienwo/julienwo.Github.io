# julienwo.Github.io

[World Bank Open Data](https://data.worldbank.org/) - A site with free, open access to global development data. Could be used to make a global map demonstrating differences in the world's economies. 

[R and Data Mining Textbook](https://cran.r-project.org/doc/contrib/Zhao_R_and_data_mining.pdf) - Written by Yanchang Zhao in 2013. A book that introduces Data Mining and how you can use R to search for interesting information within data. I was initially interested in this book because I did not know what Data Mining was and it seemed like a good introduction to the field. From the intro chapter, data mining seems like an important tool to use to find the important aspects of a dataset.
